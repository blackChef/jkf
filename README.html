Jkf

A javascript animation library that use css-keyframes-animation syntax.

Keyframes

This is a keyframes rule that jkf use:

{
    0: { left: 0, borderRadius: 0 },
    0.5: { left: '30px' },
    1: { left: '100px', borderRadius: '50%' }
}

It's a plain javascript object. It looks like a css keyframes rule. Notice that the key is a decimal instead of percentage, the property name is in camel case.
